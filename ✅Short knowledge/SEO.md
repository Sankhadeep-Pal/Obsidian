# Search Engine Optimization

**SEO in HTML** (Search Engine Optimization) refers to optimizing your HTML code so search engines can better understand, index, and rank your web pages. It plays a crucial role in improving your site's visibility in search engine results.

---

## üîç **Key HTML Elements for SEO**

Here are the most important HTML elements for on-page SEO:

|HTML Element|Purpose & Best Practices|
|---|---|
|`<title>`|Sets the page title shown in search results. Should be unique, descriptive, and under 60 characters.|
|`<meta name="description">`|Provides a summary of the page content. Often used as the snippet in search results. Max ~155 characters.|
|`<meta name="robots">`|Controls if the page should be indexed (`index/noindex`) and if links should be followed (`follow/nofollow`).|
|`<h1>` to `<h6>`|Heading tags organize content. Use one `<h1>` per page, followed by structured subheadings.|
|`<a href="...">`|Anchor tags with descriptive text improve navigation and keyword relevance.|
|`<img alt="...">`|`alt` attributes help with image SEO and accessibility. Use meaningful descriptions.|
|`<strong>`, `<em>`|These tags give importance to text (e.g., keywords), which can influence SEO slightly.|
|`<link rel="canonical" href="...">`|Helps prevent duplicate content issues by specifying the preferred URL.|
|`<meta charset="UTF-8">`|Ensures proper character encoding, helping search engines display content correctly.|
